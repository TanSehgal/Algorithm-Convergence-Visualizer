# 🚀 Algorithm Convergence Visualizer

📊 An interactive web-based tool to visualize the **convergence of numerical algorithms** used for solving mathematical equations.

---

## 🧠 Implemented Algorithms

🔹 Bisection Method  
🔹 Regula-Falsi Method  
🔹 Newton-Raphson Method  
🔹 Secant Method  
🔹 Fixed Point Iteration  

---

## 📈 Features

✨ Interactive function input  
📊 Convergence graph visualization  
📋 Iteration tables  
⚡ Real-time computation  
🧮 Supports polynomial & transcendental equations  

---

## 🛠 Tech Stack

🐍 Python  
🌐 Flask  
📊 Plotly.js  
🔢 NumPy  
📐 SymPy  
🎨 HTML / CSS / JavaScript
